## Installation

1. Clone this repository:
```
   git clone https://github.com/aldojunior990/menu-api.git
```
2. Install dependencies with Maven.

3. Install Docker in https://www.docker.com/get-started/

4. Run:
```
   docker-compose up
```

## Endpoints
```
 POST auth/login - Login into the app.
 
 POST auth/register - Register into the app.
 
 POST foods/create - Creates a new dish for a specific user's menu.
 
 GET foods/get - Returns the entire menu of an authenticated user.
```
